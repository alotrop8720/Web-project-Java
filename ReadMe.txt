Запуск проекта через Tomcat, war файл.
Доп либы: hsqldb-2.4.1; jstl-1.2; Если возникнут проблемы с Tomcat их нужно будет добавить к библиотекам.

 Как пользоватся:
    Сверху в меню можно выбрать к какой таблице перейти.
    Добавить: нажать на кнопку добавления внизу под таблицей.
    Изменить: нажать на поле таблицы которое хотите изменить.
    Удалить: нажать на кнопку удалить соотвествущей строке.
 Я не делала валидацию, так что данные вводить в соотвествии с типом поля.
